# getMatrix

Напишите функцию `getMatrix` создающую матрицу заданного размера, заполненную значениями `null`.

Пример:

```javascript
getMatrix(4, 6);
/*
[
    [null, null, null, null],
    [null, null, null, null],
    [null, null, null, null],
    [null, null, null, null],
    [null, null, null, null],
    [null, null, null, null],
]
*/
```
