# getTranspose

Транспонированная матрица - матрица в которой строки и столбцы поменялись местами.

Напишите функцию `getTranspose` принимающую матрицу, и возвращающую ее транспонированную версию.

Пример:

```javascript
getTranspose([
	[1, 2, 3, 4],
	[5, 6, 7, 8],
]);
/*
[
    [1, 5],
    [2, 6],
    [3, 7],
    [4, 8],
]
*/
```
